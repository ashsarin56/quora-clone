now we have decided the schema and we know that user will login frequently and wants
longer sessions so we will use JWT auth for the validation of the user 

so the flow is like 
we will receive 
    name
    email
    password
from req.body();
if email already exists then the user is already registered and they should login.
if its not registered
    REGISTER
    then bcrypt the password 
    create new user in db
    generate JWT token
    send token to the user 
    then for each request requested by user 
    if the http request contains a token header 
    where the hash generated by the same JWT key is equal 
    then the request is processed and hence we authenticate every request 

    LOGIN
    recieve email and password from req.body();
    find user by email if not found return error
    Compare submitted password with hashed password
    if wrong then return error
    Generate a JWT Token
    Send token and user data back to frontend

    so structure is like 
    controllers
        authcontroller -> logic is implemented here
    routes
        authroute.js -> URL mapping is implemented here
    
    now the auth is implemented and we want to verify the user at each requet so that only authenticated 
    user can access some of the routes these routes which only can be accessed by authenticated user are
    called PROTECTED routes
    now we need a middleware to verify this 
    so we will implement a authmidleware to protect the routes

    now middle ware is done and we need to design the question/answer routes 
    so they will have multiple methods
    so while posting a answer or a question we need to verify user
    and we need details of author too
    so we will fetch the details of author from server only 
    (we got data in the payload )
    and then we can deal with our question route 
    we can freely make them public or protected 

    so the route structure for question looks like

    GET    /api/questions      -> getAllQuestions  (public) 
    GET    /api/questions/:id  -> getQuestion      (public)
    POST   /api/questions      -> createQuestion   (protected)
    DELETE /api/questions/:id  -> deleteQuestion   (protected)
